Software Projects

Here are some of the computational tools and software projects I've developed or contributed to:

CryoJAM

Automating protein homolog fitting in medium resolution cryo-EM density maps. This tool streamlines the process of fitting known protein structures into cryo-EM maps.
GitHub Paper
Python Cryo-EM Bioinformatics

Multi-omics Data Fusion

A data-fusion approach for diagnosing developmental dyslexia using multi-omics datasets. Combines multiple biological data types for improved diagnostic accuracy.
GitHub Paper
R Machine Learning Multi-omics

TR-XES Data Visualization

Time-resolved X-ray emission spectroscopy data visualization tools for studying transient states in Photosystem II.
GitHub Poster
Python Data Visualization X-ray Spectroscopy

PolypNET

Lightweight UNET++ architecture for identifying and segmenting cancerous polyps from colonoscopy images with high accuracy.
GitHub
Python Deep Learning Computer Vision UNET++
